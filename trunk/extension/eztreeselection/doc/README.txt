Datatypes
=========
eztreeselectiontype
-------------------
This datatype lets you create a tree in the class-level edition interface. 
The leaves of this tree will be selectable, as options were in the legacy 'ezselection' datatype.

When editing an object having such an attribute, the user will be prompted to select one or several options.

It supports multiple/single selection.
